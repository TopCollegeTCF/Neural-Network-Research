# Исследование генеративных нейросетей: Генерация кода (Платформер)

Добро пожаловать в мой учебный курс/эксперимент по анализу возможностей генеративных нейросетей.
Основная цель: проверить, насколько хорошо разные модели справляются с написанием рабочего кода при жестких ограничениях.

## Легенда и задача

В рамках исследования я даю нейросетям одну и ту же задачу:
> "создай простую игру платформер на html / css / js - лимит 200 строк вообщем"

Это позволяет оценить не только "магию" искусственного интеллекта, но и его способность укладываться в технические требования (лимит строк), оптимизировать код и выдавать играбельный результат.

## Структура репозитория

В корне репозитория в каждой ветке находятся:
*   `README.md` - этот файл, общее описание проекта.
*   В репозитории будут Issues с описанием каждого шага эксперимента названные именем ветки и номером шага.
*   Папки с названиями нейросетей (например, `ChatGPT4/`, `Claude/`, `GigaChat/`). Внутри каждой папки:
    *   `index.html` - рабочий код игры, сгенерированный сетью.
    *   `README.md` - отчет по конкретной сети: введенный промпт, анализ работы, плюсы и минусы, ссылка на видео/скриншот (если есть).

## Результаты и анализ

В этом разделе (или в отдельных файлах по папкам) я буду публиковать выводы:
*   Какая нейросеть написала самый красивый код?
*   Какая игра оказалась самой играбельной?
*   Кто нарушил лимит в 200 строк, а кто вписался идеально?
*   Личное мнение о каждой модели.

## Как запустить

1.  Перейдите в интересующую вас папку (например, `Claude/`).
2.  Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).
3.  Играйте! (Обычно управление осуществляется стрелочками или WASD).

## Цели курса

1.  Собрать сравнительную характеристику популярных нейросетей (зарубежных и российских).
2.  Понять, умеет ли ИИ следовать нефункциональным требованиям (лимит строк).
3.  Создать базу знаний "какой промпт к какой сети писать для получения кода".
