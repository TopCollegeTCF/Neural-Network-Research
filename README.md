### Инструкция: Какие нейросети попробовать

> [!NOTE]
> Для чистоты эксперимента и получения разнообразных результатов, я разделил нейросети на категории. Рекомендую начать с первой категории, > так как они лучше всего подходят для написания кода.

> [!IMPORTANT]
> #### Чек-лист для выполнения задания
>
> 1.  **Выбор:** Выберите 2-3 нейросети из списка выше или найдите другие.
> 2.  **Запрос:** Открываете выбранную нейросеть.
> 3.  **Промпт:** Копируете текст точно: *"создай простую игру платформер на html / css / js - лимит 200 строк вообщем"*.
> 4.  **Фиксация:** Копируете полученный код.
> 5.  **Сохранение:**
>    *   Создаете в своем репозитории папку `name` (name - название нейросети).
>    *   Создаете файл `index.html`. Вставляете туда код. Если код разбит на три части (CSS внутри `<style>` и JS внутри `<script>`), то > > всё отлично.
>    *   Проверяете, работает ли игра (открываете `index.html` в браузере).
